Linguagem de Programação

Na Matéria de Linguagem de programação vimos logica de programação, como subir arquivos no GitHub, conceitos do HTML e CSS, fundamentos do JavaScript.

Para aprender e consolidar o conteúdo foram passados diversos projetos, os quais irei apresentar a seguir.

Conteúdo

HTML e CSS

  Nas aulas de HTML inicialmente nós vimos as principais tags do HTML, logo em seguida vimos como estruturar um site usando essas tags. Aprendemos como criar títulos, pular linhas, colocar imagens e criar links. No decorrer das aulas também aprendemos HTML semântico, a sua importância para que o código do site fique mais limpo e se caso outras pessoas vejam, consigam entender.

  Nas aulas de CSS nos foi apresentado como integrar o CSS no HTML, com isso aprendemos como estilizar nosso site, adicionando cores e elementos, manusear contêiners de elementos, aprendemos posteriormente como fazer elementos se moverem ao passar o mouse por cima.

Linguagem JavaScript

  Nas aulas de JavaScript inicialmente vimos conceitos básicos de todas as linguagens como: variáveis, tipos de dados, sitaxe, operadores lógicos, operadores aritméticos. Também vimos como realizar como operações matemáticas e utilizar estruturas de decisão (as estruturas de decisão são condições criadas e checadas dentro do código, como por exemplo: if/else e switch) e estruturas de repetição (as estruturas de repetição são loops criados no código, como por exemplo: for, while e do while), vimod também funções, arrays etc.

  Após aprender os conceitos básicos do JavaScript aprendemos como integrar ele com o HTML e utilizá-lo para melhorar nosso site, já que quando usamos o JavaScript conseguimos criar estilos mais elaborados para utilizar em nossos sites.

  Junto com as aulas de JavaScript aprendemos também o HTML DOM (Document Object Model), o DOM é construído como uma arvore de objetos, e assim facilitando a interação do JavaScript com o HTML.

Site de Receitas

Esse projeto nos permitiu utilizar pela primeira vez elementos do HTML. Com ele aprendemos organizar os conteúdos dentro do site e a utilizar tags básicas como por exemplo o < h1 >, e também a criar listas ordenadas e não ordenadas. Aprendemos também como adicionar fotos a uma pagina HTML e a criar um link que leva você a uma nova página.

Currículo

Com esse projeto aprendemos um pouco mais sobre a formatação das telas utilizando o CSS. Nesse projeto aprendemos a integrar o CSS ao HTML e assim conseguir fazer uma formatação mais completa do site que estava criando. Nesse projeto aprendemos a usar elementos de cor e formatação de div para organizar todos os elementos introduzidos no site, também utilizamos novamente o recurso de linkar uma pagina a outra.

Html 5

Nesse projeto aprendi a importância do HTML semântico e como utiliza-lo corretamente, também aprendi a criar novos elementos como a colocar sobras, como utilizar o hover para fazer com que alguns elementos mudem ao passar o mouse em cima. Também aprendi a criar botões e caixas de texto.

Folha de Pagamento

Esse projeto consistiu em cria um site que calcula os descontos da folha de pagamento. Esse foi uns dos primeiros projetos que utilizamos o JavaScript integrado ao HTML. Com esse projeto aprendemos a integrar o JS ao HTML e também como utiliza-lo para manipulação de dados.

Conversor

Nesse projeto aprendemos a criar uma pagina com entradas para textos HTML e JavaScript e rodar eles com uma formatação CSS ja pronta.
